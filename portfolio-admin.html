Portfolio Code Generator (සරල කළ)

මෙය භාවිතා කරන ආකාරය

මෙම සරල කළ panel එක මගින්, ඔබගේ `portfolio.html` පිටුවට අවශ්‍ය HTML කේතය පහසුවෙන් ජනනය කරගත හැක.

  1. පියවර 1: ඔබගේ පින්තූර වල Links (Image URLs) එකක් යට එකක් පහත "Image URLs" කොටුවේ paste කරන්න. (ඔබට එක link එකක් හෝ කිහිපයක් වුවද paste කල හැක).
  2. පියවර 2: එම පින්තූර සියල්ලටම අදාල "Category" එක තෝරන්න.
  3. පියවර 3: "Generate Bulk Code" බොත්තම ඔබන්න.
  4. පියවර 4: පහත ජනනය වන සම්පූර්ණ HTML කේතය "Copy" බොත්තම මගින් copy කරගන්න.
  5. පියවර 5: ඔබගේ `portfolio.html` ගොනුව විවෘත කර, `` යන්නට මදක් ඉහලින්, මෙම කේතය paste කරන්න.
  6. පියවර 6: `portfolio.html` ගොනුව save කර, GitHub වෙත නැවත upload (commit/push) කරන්න.
